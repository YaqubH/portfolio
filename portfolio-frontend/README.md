# Portfolio Frontend

React + Vite + Tailwind frontend for the AI Portfolio Assistant.

## Requirements

- Node.js 18+

## Setup

```bash
cd portfolio-frontend
npm install
```

## Run

```bash
npm run dev
```

Opens at **http://localhost:5173**

## Notes

- The chatbot calls the backend at `http://localhost:8000/chat`
- Start the backend first or the chatbot will show a connection error
- See `portfolio-backend/README.md` for backend setup
